How it works

This simple bridge:

  • Automatically starts the JavaScript backend when needed
  • Forwards MinAI requests to the backend
  • Returns responses to HerikaServer

Troubleshooting

If something's not working:

  • Check that Node.js is installed
  • Make sure port 8080 is available
  • Check server error logs
