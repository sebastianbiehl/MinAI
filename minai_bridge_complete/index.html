MinAI Bridge Plugin

This plugin bridges MinAI functionality from PHP to a clean JavaScript implementation.

Features

  • Self-Narrator: Internal player thoughts and commentary
  • Translation: Converts casual speech to character dialogue
  • Dungeonmaster: NPCs treat events as canonical

How It Works

  1. HerikaServer loads this bridge plugin
  2. Bridge automatically starts JavaScript backend server
  3. All MinAI requests are forwarded to the JavaScript backend
  4. Clean JavaScript modules process the requests
  5. Responses are applied back to HerikaServer

Benefits

  • Eliminates PHP function redeclaration errors
  • Clean, maintainable JavaScript implementation
  • Better error handling and logging
  • Easy to configure and extend
